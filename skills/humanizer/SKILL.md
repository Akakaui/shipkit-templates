# HUMANIZER SKILL

Last updated: 2026-06-26
Version: 1
Scope: ALL content-producing agents — final step before output

## PURPOSE

Strip every trace of AI-generated writing patterns from output.
The result must read like a real person wrote it. Not "AI with
human touches." Actually human.

## THE PROBLEM

AI writing has recognizable patterns:
- Overuse of "delve," "landscape," "realm," "tapestry"
- Starting sentences with "In a world where..."
- Excessive em dashes and parenthetical asides
- Perfectly parallel sentence structures
- Hedging language ("it's worth noting," "it's important to mention")
- Lists that are always exactly 3-5 items
- Conclusions that summarize everything just covered
- Transitional phrases that sound like a textbook

## THE FIX

### Step 1: Word Replacement

Replace these words/phrases immediately:

  "delve" → "dig into" or "look at"
  "landscape" → remove or use "space" or "world"
  "realm" → "area" or "space"
  "tapestry" → remove entirely
  "leverage" → "use"
  "utilize" → "use"
  "facilitate" → "help" or "run"
  "implement" → "build" or "set up"
  "subsequently" → "then"
  "furthermore" → "also" or start new sentence
  "in conclusion" → just conclude
  "it's worth noting" → remove
  "it's important to mention" → remove
  "in today's fast-paced" → remove
  "game-changer" → describe what it actually does
  "synergy" → remove
  "holistic" → "full" or "complete"
  "paradigm" → remove
  "ecosystem" → "stack" or "setup"

### Step 2: Sentence Structure Check

- No two consecutive sentences should start the same way
- Vary sentence length: short, medium, short, long, short
- If three sentences in a row are the same length, break one
- Kill any sentence that says the same thing as the one before it

### Step 3: Personality Injection

Add at least one of these per piece:
- A specific detail only a human would know
- A casual aside or aside
- A short, punchy sentence that breaks the rhythm
- A direct address to the reader ("you" not "one")
- An opinion stated without hedging

### Step 4: Read Aloud Test

Read the final output. If it sounds like:
- A blog post → rewrite
- A LinkedIn generic post → rewrite
- A textbook → rewrite
- A speech → rewrite

It should sound like:
- A smart friend explaining something over coffee

## OUTPUT CHECKLIST

Before delivering any content, verify:

- [ ] Zero instances of banned words from Step 1
- [ ] No two consecutive sentences start the same way
- [ ] At least one specific detail or personal touch
- [ ] No hedging language
- [ ] First line hooks immediately
- [ ] Last line has a clear ending or CTA
- [ ] Reads naturally when spoken aloud
